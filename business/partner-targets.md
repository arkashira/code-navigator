# partner-targets.md

## Partner Integration Roadmap for Code Navigator

### Target Integrations

1. **GitHub API**
   - **Rationale**: As a primary platform for code hosting and collaboration, integrating with GitHub allows users to directly navigate and analyze their repositories within Code Navigator.
   - **Free-Tier Limits**: 2000 requests per hour.
   - **Integration Effort**: Medium (M)
   - **Value-Add**: Enables users to pull in their codebases seamlessly, enhancing the user experience and reducing friction in setup.

2. **GitLab API**
   - **Rationale**: Similar to GitHub, GitLab is widely used for version control and CI/CD. This integration will cater to users who prefer GitLab as their code repository.
   - **Free-Tier Limits**: 500 requests per hour.
   - **Integration Effort**: Medium (M)
   - **Value-Add**: Allows users to navigate their GitLab repositories, expanding the tool's usability across different platforms.

3. **Bitbucket API**
   - **Rationale**: Bitbucket is another popular code hosting service. Integrating with Bitbucket will broaden Code Navigator's appeal to a larger audience.
   - **Free-Tier Limits**: 1000 requests per hour.
   - **Integration Effort**: Medium (M)
   - **Value-Add**: Facilitates access to Bitbucket repositories, making it easier for teams using this platform to adopt Code Navigator.

4. **Slack API**
   - **Rationale**: Integrating with Slack allows users to receive notifications and updates about code changes and navigation insights directly in their communication channels.
   - **Free-Tier Limits**: 10 integrations and 5 apps.
   - **Integration Effort**: Small (S)
   - **Value-Add**: Enhances collaboration by keeping team members informed about codebase changes in real-time.

5. **Jira API**
   - **Rationale**: Many development teams use Jira for project management. Integrating with Jira can help link code changes to specific tasks or issues, providing context to developers.
   - **Free-Tier Limits**: 10 users.
   - **Integration Effort**: Medium (M)
   - **Value-Add**: Helps users track code changes related to project tasks, improving workflow efficiency.

6. **CircleCI API**
   - **Rationale**: As a CI/CD tool, integrating with CircleCI allows users to see the status of their builds and deployments in relation to their codebase navigation.
   - **Free-Tier Limits**: 2,500 build minutes per month.
   - **Integration Effort**: Medium (M)
   - **Value-Add**: Provides insights into the CI/CD pipeline, helping developers understand the impact of code changes on deployments.

7. **Trello API**
   - **Rationale**: Trello is widely used for task management. Integrating with Trello can help users manage their development tasks alongside code navigation.
   - **Free-Tier Limits**: Unlimited boards, lists, and cards for personal use.
   - **Integration Effort**: Small (S)
   - **Value-Add**: Facilitates task management in conjunction with code navigation, improving overall project organization.

8. **Zapier**
   - **Rationale**: Zapier allows integration with numerous apps, enabling users to create automated workflows between Code Navigator and other tools they use.
   - **Free-Tier Limits**: 5 Zaps and 100 tasks per month.
   - **Integration Effort**: Large (L)
   - **Value-Add**: Expands functionality by allowing users to automate repetitive tasks, enhancing productivity.

### Prioritization
- Focus on integrations with GitHub, GitLab, and Bitbucket first due to their direct relevance to code navigation.
- Follow up with Slack and Jira to enhance collaboration and project management capabilities.
- Consider CircleCI and Trello for CI/CD and task management integrations to provide a comprehensive development experience.
- Finally, explore Zapier for broader automation possibilities, which can significantly enhance user workflows. 

### Revenue-Share Opportunities
- Investigate affiliate programs for GitHub, GitLab, and Bitbucket, as these platforms often have partner programs that could provide revenue-sharing opportunities.
